What is phishing?

A Phishing website is a type of fraudulent site designed to deceive visitors into disclosing sensitive information, such as login credentials, financial details, or personal data. These sites mimic legitimate websites, such as banks, social media platforms, or online retailers, to trick users into believing they are interacting with a trusted entity.

URL - Uniform Resource Locator

URL is used to address web pages.
  • A domain name serves as the address of a website, identifying the server that hosts the web page.
  • It consists of a registered second-level domain (e.g., "exampleurl") and a top-level domain (e.g., ".com"). This part of the URL is regulated, as it must be registered with a domain name registrar.
  • A hostname includes a subdomain (third-level domain, e.g., "www") and the main domain name. The URL may also include path and file components.

Parts of a URL that a phisher can manipulate

Phishers have complete control over the subdomain and can set it to any value. The phishers can also manipulate the file and path components. Both the subdomain and path are fully controllable by phishers (e.g., "fake.example.com/path").

Cybersquatting

Registering "appleincorporation.com" with the intention of selling it to Apple Inc. for a profit.

Cybersquatting refers to the practice of registering, trafficking in, or using a domain name with the intent to profit from the goodwill of a trademark belonging to someone else. The cybersquatter typically holds the domain name ransom, offering to sell it at an inflated price to the trademark owner or a third party.

Typosquatting

Registering "goggle.com" to trick users who accidentally type an extra "g" instead of "o" when trying to visit "google.com".

Typosquatting, also known as URL hijacking, is a form of cybersquatting that involves registering domain names that are common misspellings or variations of well-known websites. The intent is to capitalize on typographical errors made by users when entering a website address into their browser.
